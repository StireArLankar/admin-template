# Template for admin panel

- `prisma`
- `tailwind (tailwind-variants + typewind)`
- `trpc`
- `express`
- `vite`
- `@tanstack/react-router` and `@tanstack/react-table`
- `@radix-ui`

## Commands

### Install dependencies with pnpm

```bash
pnpm i
```

### Run dev client and server

```bash
pnpm dev
```
